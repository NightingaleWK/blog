Git Pull 强制拉取步骤

2021-12-21 git

生成公钥

git fetch –all
git reset –hard origin/main
git pull

git fetch –all && git reset –hard origin/main && git pull


注意: ``git reset --hard origin/分支代码``, 一般都是主干,若有多期工程, 需自己更改
